# Medisphere - Healthcare Management System

A web-based Healthcare Management System built using Spring Boot, Spring Security, Thymeleaf, and MySQL. It allows admin users to manage patients, doctors, and appointments through a secure, role-based interface.

## Features

### Admin Authentication
- Secure login using Spring Security
- Role-based access control (Admin/Doctor)

### Patient Management
- Add new patients
- View all patients
- Update patient details
- Delete patient records

### Doctor Management
- Add new doctors
- View all doctors
- Update doctor details
- Delete doctor records

### Appointment Management
- Book appointments (linking patients and doctors)
- View all appointments
- Update appointment status (Confirmed / Completed)
- Cancel appointments

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Spring Boot 3.3.4 |
| Security | Spring Security |
| Frontend | Thymeleaf, HTML, CSS |
| Database | MySQL |
| ORM | Spring Data JPA / Hibernate |
| Build Tool | Maven |
| IDE | VS Code |
